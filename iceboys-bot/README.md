# 🧊 ICEBOYS MONETIZER BOT
Automated Telegram subscription management with blockchain verification.

## 🚀 Features
- **ETH Payments:** Automatic 0.01 ETH verification via Alchemy.
- **Referral System:** Viral growth via +2 days free rewards.
- **Auto-Invite:** Instant private channel access upon payment.

## 🛠 Setup
1. Clone this repo.
2. Fill your `.env` with your Telegram Token and Alchemy URL.
3. Deploy to Render as a Background Worker.
